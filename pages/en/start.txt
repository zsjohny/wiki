Will Support Function
-------------------------


----------


 * History Cleanup [plugin:clearhistory](https://www.dokuwiki.org/plugin:clearhistory)

 * Insert table  [plugin:edittable](https://www.dokuwiki.org/plugin:edittable) 

 * Create a list of navigation [plugin:indexmenu](https://www.dokuwiki.org/plugin:indexmenu) 

 * Add comments area [plugin:discussion](https://www.dokuwiki.org/plugin:discussion)

 * Add New Page [plugin:addnewpage](https://www.dokuwiki.org/plugin:addnewpage)

 * Searchindex [plugin:searchindex](https://www.dokuwiki.org/plugin:searchindex) 

 * Translation [plugin:translation](https://www.dokuwiki.org/plugin:translation) 

 * Create a simple navigation tree based on namespaces [plugin:simplenavi](https://www.dokuwiki.org/plugin:simplenavi)

 * Backup page, revision history, media, configuration, etc. All the content [plugin:backup](https://www.dokuwiki.org/plugin%3Abackup) 

 * Create a simple, checkbox based ToDo List [ToDo Plugin](https://www.dokuwiki.org/plugin%3Atodo)

 * Present a table of contents of the pages of a selected namespace [Note Plugin](https://www.dokuwiki.org/plugin%3Anote)

 * Show which groups exist, what their namespace and page permissions are and which users are in which groups. [Permission Info Plugin](https://www.dokuwiki.org/plugin:permissioninfo)
